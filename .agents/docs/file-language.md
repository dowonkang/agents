# File Language Policy

Files intended for agents must be written in English.

Examples:

- `AGENTS.md`
- Agent instructions
- Internal agent guidance
- Implementation notes for agents

Files intended for users may be written in any language the user requests.

Examples:

- User guides
- README sections for end users
- Tutorials
- Release notes

If a file has both agent-facing and user-facing content, ask the user which language policy applies before creating or editing it.
